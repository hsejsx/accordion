웹 표준과 웹 접근성을 준수하는 HTML 정보 구조설계로 웹사이트 퍼블리싱을 해오고 있습니다.
특히 웹 표준을 위해 HTML5에서 권장하는 마크업 방식과 시멘틱 태그를 사용해 전체 레이아웃을 설계합니다.
특히, 웹 사이트를 퍼블리싱 하기 전에 '기능정의서'를 만들고 기능정의서에 전체적인 폰트 서식, 컬러 가이드, 사이즈 정보, 상태 디자인, HTML 와이어프레임까지 꼼꼼하게 미리 정리해 놓고 체계적인 퍼블리싱 작업을 수행합니다.

단순히 디자인을 구현하는 것에 멈추는 것이 아닌, 어떤 상황에서든 UI가 망가지지 않도록 다양한 상황들을 생각하며 처리합니다.

네비게이션, 탭 메뉴, 아코디언 등 웹 퍼블리싱에 반드시 필요한 상호작용을 위한 제이쿼리는 플러그인을 사용하지 않고 순수한 제이쿼리 코딩으로 제작합니다. 바닐라 자바스크립트도 사용 가능하며, 주로 제이쿼리와 병행해서 활용합니다.
